# Tanjai POS - User Manual

## For Cashiers

### 1. Opening the Shift

1. Login with your PIN.
2. Check the **Cash Drawer** balance.
3. Tap **"Open Shift"** on the dashboard.

### 2. Taking Orders

1. Tap on the **Menu** tab.
2. Select items to add to the cart.
   - For customizable items, select options (e.g., Spicy Level).
3. Review the **Cart** on the right side.
4. Tap **"Place Order"** or **"Checkout"**.

### 3. Processing Payments

1. Select payment method:
   - **Cash**: Enter amount received. System calculates change.
   - **QR PromptPay**: Show QR code to customer. System auto-confirms.
2. Print receipt if requested.

### 4. Closing the Shift

1. Go to **Settings > Shift**.
2. Count cash in drawer and enter the amount.
3. Tap **"Close Shift"**.
4. Print the "Z-Report" for the owner.

---

## For Owners

### 1. Menu Management

- Go to `[YourShopURL]/admin/menu`.
- **Add Item**: Click "+ Add Item", upload photo, set price.
- **Edit Item**: Click on any item to update availability (In Stock / Out of Stock).

### 2. Dashboard & Reports

- **Live Sales**: View real-time sales on the main dashboard.
- **History**: filtering orders by date range.

### 3. Staff Management

- Create accounts for new staff in the **Staff** section.
- Assign roles (Cashier / Manager).
