# How to Use Smart QR Generator

## Accessing the Dashboard
1. Log in to your Owner Dashboard.
2. Navigate to the **QR Generator** tab in the sidebar.

## 1. Static Query (For Tables)
Use this mode to create permanent QR codes for your tables.

1. Select **"Static"** from the tabs.
2. Choose the **Table Number** from the dropdown menu (1-50).
3. The system immediately generates a QR code.
4. Click **"Print Table Label"** to download a high-resolution version for your thermal printer.
5. Stick this QR on the table. Customers scanning it will see the full menu.

## 2. Dynamic Query (For Payments/Orders)
Use this mode to accept payments or specific orders.

1. Select **"Dynamic"** from the tabs.
2. Enter the **Order Amount (THB)**.
3. Click **"Generate Payment QR"**.
4. The system creates a unique QR code valid for **15 minutes**.
5. Show this QR to the customer. When they scan it, they will be directed to the payment confirmation page.
6. Once scanned/paid, the QR expires automatically.

> [!NOTE]
> Dynamic QRs are One-Time Use only for security. Do not print these for reuse.
