# DOC-0002: SOP - Opening & Closing Shift

**Objective**: Ensure the store is ready for service and secure at closing.

### 1. Device Check
- [ ] Turn on all KDS Tablets and POS Terminals.
- [ ] Verify Wi-Fi connection (Network: `TanJai_POS_Secure`).
- [ ] Check Receipt Printer paper roll. Reserve roll must be nearby.

### 2. Cash Drawer
- [ ] Count opening float (Standard: 2,000 THB).
- [ ] Log amount in "Cash In" function on POS.

### 3. Inventory Spot Check
- [ ] Verify critical stock (e.g., Pork, Ice, Cups).
- [ ] If low, trigger "Urgent Order" in Inventory App.

## ðŸŒ™ Closing Shift (Night)

### 1. Financial Reconciliation
- [ ] Print "Z-Report" (Daily Sales).
- [ ] Count Cash Drawer.
- [ ] Deposit cash into Safe Drop. Enter amount in "Cash Out".
- [ ] Staple Z-Report with Credit Card settlement slip.

### 2. Kitchen Shutdown
- [ ] Clear all "DONE" tickets on KDS.
- [ ] Turn off Tablets (Screen off only, do not power down unless charging).

### 3. Security
- [ ] Upload Offline Orders: Go to **Settings > Sync Status**. Ensure "Pending" is 0.
- [ ] Lock all doors.
