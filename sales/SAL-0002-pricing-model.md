# SAL-0002: Pricing Model Proposal

**Strategy**: Hybrid Model to lower barrier to entry (SME) while capturing value from high-volume shops (Pro).

## Option A: "We Grow Together" (SME / Starter)
**Best for**: Street Food, Small Cafes (< 10 Tables).

- **Monthly Fee**: **0 THB** (Free Forever).
- **GP Share**: **2.5%** on Orders processed via QR.
- **Hardware**: BYOD (Bring Your Own Device). Use any Android/iPad.
- **Features**:
    - Unlimited Menu Items.
    - Basic Inventory.
    - Standard Support (Email).

## Option B: "Professional Power" (Established)
**Best for**: Busy Restaurants, Franchises.

- **Monthly Fee**: **990 THB / Month**.
- **GP Share**: **0%** (Transaction fees from Bank only).
- **Hardware**: Dedicated Sunmi V2 Device (Rental +300/mo).
- **Features**:
    - **Advanced BOM Inventory** (Ingredient Cutting).
    - **Astro-Promo Engine**.
    - **Priority Support** (Line OA Live Chat).
    - Multi-Branch Management.

## Option C: "Enterprise Custom"
**Best for**: Food Courts, large Chains (> 10 Branches).

- **Fee**: Custom Quote.
- **Features**: Whitelabel App, API Access, Sap Integration.
